```javas
1、创建项目
2、初始化 git init 
3、添加项目到暂存区 git add 文件夹名称
4、提交文件夹到本地库 git commit -m "first commit" 文件夹名称
5、查看远程库 git remote -v
6、创建远程库别名 git remote add gitReactProject 远程库的地址
7、本地库中项目推送到远程库  git push 远程库地址 当前所处分支名称
8、切换分支操作 git checkout 分支名称
9、创建新分支  git branch 分支名称
10、合并分支  git merge 分支名称A   这样分支A就会被合并到分支B中
```
